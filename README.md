# 2025gsc_MiaKozaki
2025年ゼミ論
## テーマ
プロ野球の勝敗と試合展開が都市回遊行動に与える影響分析

## 概要/Abstract
本研究は、プロ野球の試合結果、特にその「興奮度（ドラマ性）」が、観戦客の試合終了後の行動範囲と消費行動に与える影響を、位置情報データ（GPSログ）を用いて定量的に分析することを目的とする。
既存研究の「安全」や「観客数」といった視点から一歩踏み込み、心理学的な知見（BIRGing効果など）を応用し、「感情の強さ」が「物理的な移動距離」にどのように変換され、地域経済圏を伸縮させるかを検証する。具体的には、QGISを用いた差分ヒートマップ分析により、条件統制された「劇的勝利」の日と「大敗」の日を比較し、勝利によって純増する人流エリアを特定する。
本研究の成果は、スポーツイベントにおける感情に呼応する「動的なエリアマネジメント」への応用を可能にし、都市運営とマーケティング戦略に新たなデータ視点を提供する。


## 研究の背景
従来のスポーツ経済学研究は、スタジアム建設の経済波及効果や年間の観客数といったマクロ的な指標に偏重しており、試合終了直後というミクロな時間帯における観客の具体的な消費行動や移動パターンについては十分に解明されていない。特に、観客が持つ「感情の質（喜び、落胆など）」が、その後の都市空間における消費行動に与える影響は、科学的に定量化されていないのが現状である。
本研究は、以下の3つの主要な学術領域の「隙間」を埋めることを目指す。
- スポーツ経済学（例：萩原ほか, 2018）： マクロな分析に終始し、ミクロな回遊行動を捉えられていない。
- スポーツ心理学（例：松井, 2011）： BIRGing効果（栄光浴）などの心理的な変化を捉えているが、それが「地図上の移動距離」という物理的行動にどう変換されるかは不明。
本研究は、これらを統合し、「感情（ドラマ性）」と「都市空間での移動（物理的距離）」の相関を分析することで、学術的貢献を果たす。

## 研究目的
- プロ野球の試合結果と試合展開が、観客の「興奮度（ドラマ性）」を介して、試合終了後の回遊範囲と滞在時間に与える影響を定量的に明らかにする
- 最終的に、その結果に基づき、スポーツイベントにおける感情に呼応する「動的なエリアマネジメント手法」を提言

## リサーチクエスチョン
ホームチームの「試合のドラマ性（興奮度）」は、観客の足を何メートル遠くまで、どのような属性のエリア（例：ローカルな個店 or 駅前チェーン店）へ誘引するのか？

## 仮説
「試合の感動レベル（ドラマ性）が高いほど、観客のポジティブ感情が持続・増幅し、スタジアム周辺のローカルなエリアへの回遊距離を伸ばす」という相関仮説を立てる。

## 研究方法

本研究では、QGISを用いた空間分析により、以下の変数の関係性を検証する。
- 独立変数（原因）： 試合の興奮度指数（点差、逆転回数、サヨナラなどの客観指標）
- 従属変数（結果）： 試合終了後30分間の観客の平均移動距離、滞留地点の属性（飲食店の密度・種類）

### データ選定と条件統制（最大のリスクヘッジ）

研究の信頼性を確保するため、以下の厳しい条件を設定して比較対象試合を抽出する。
- 比較対象の抽出: 曜日（平日ナイター）、天候（晴天）、試合終了時刻が揃った試合同士をリストアップする。
- ノイズ除去: 実データ解析において、試合時間帯にスタジアム周辺に滞在していたユーザーIDのみを抽出するフィルタリングを行い、通勤客や住民のデータを排除する。

### 技術的解析プロセス（QGISを用いたPoC）

実データ解析に先立ち、解析フローの確立（PoC）を行った。
静的ポテンシャルの可視化: QuickOSMを用いて、飲食サービス（pub/restaurant）の密度をヒートマップ化し、祝杯エリアの「受け皿」を特定した。
動的解析フローの確立: 生成AIにより作成した合成GPSログデータを使用し、QGISの時系列コントローラを用いた動的流動シミュレーションが技術的に可能であることを確認した。

- 空間統計処理: 最終解析では、条件統制された「勝ち日」と「負け日」のデータに対し、差分ヒートマップ分析（勝ち日分布 ー 負け日分布）を行い、勝利によって純増した人流エリアを抽出する。

 **データ収集**  
   - 人流データ：現在、複数のデータソースを検討中  
        -  候補：G空間情報センター（1kmメッシュ）、民間アプリGPSデータ（Agoop等）、キャリア系携帯位置情報（ドコモ等）  
        - 選定方針：空間解像度（250〜500m程度）、時間解像度（30分程度）、コスト・調達期間を考慮して決定予定  
        - 必要に応じて現地観察調査で補完する 
   - 試合日程：NPB公式、DeNAベイスターズ公式  
   - 空間要因データ：OpenStreetMap、Google Mapsなど
