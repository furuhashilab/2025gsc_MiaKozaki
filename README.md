# 2025gsc_MiaKozaki
2025年ゼミ論
## テーマ
プロ野球の勝敗と試合展開が都市回遊行動に与える影響分析

## 概要/Abstract
本研究は、プロ野球の試合結果、特にその「興奮度（ドラマ性）」が、観戦客の試合終了後の行動範囲と消費行動に与える影響を、位置情報データ（GPSログ）を用いて定量的に分析することを目的とする。
既存研究の「安全」や「観客数」といった視点から一歩踏み込み、心理学的な知見（BIRGing効果など）を応用し、「感情の強さ」が「物理的な移動距離」にどのように変換され、地域経済圏を伸縮させるかを検証する。具体的には、QGISを用いた差分ヒートマップ分析により、条件統制された「劇的勝利」の日と「大敗」の日を比較し、勝利によって純増する人流エリアを特定する。
本研究の成果は、スポーツイベントにおける感情に呼応する「動的なエリアマネジメント」への応用を可能にし、都市運営とマーケティング戦略に新たなデータ視点を提供する。

## Introduction
本研究は、プロ野球の試合結果、特に「試合のドラマ性（興奮度）」が、観戦客の試合終了後の都市回遊行動にどのような影響を与えるのかを、位置情報データ（GPSログ）によって定量的に明らかにすることを目的とする。
従来のスポーツ経済学は、スタジアム建設による経済波及効果や年間観客数といったマクロ指標に焦点が当てられてきた（萩原ほか, 2018）。しかし、試合終了直後というミクロな時間帯の回遊行動や、観客の感情がその行動をどう変容させるかは十分に検証されていない。一方で、スポーツ心理学ではBIRGing（Basking in Reflected Glory）効果（松井, 2011）など、勝利がもたらす心理的高揚効果は知られているが、心理変容が「地図上の移動距離」という物理的行動にどう変換されるかについての研究は見られない。
本研究はこれらの学術領域の隙間を埋めるものであり、「感情（興奮度）」と「都市空間での移動（物理距離）」との関連に着目し、試合のドラマ性が観客の回遊範囲へ与える影響を総合的に分析する点に学術的意義がある。

研究目的：
- プロ野球の試合展開と興奮度が、観客の都市回遊距離および滞在エリアに与える影響を定量的に解明する。
- 感情に応じて変化する人流特性に基づき、スポーツイベントにおける「動的エリアマネジメント」への示唆を得る。

リサーチクエスチョン（RQ）：
ホームチームの「試合のドラマ性」は、観客の足を何メートル遠くまで、どのような属性のエリアへ誘引するのか。

仮説：
試合の感動レベル（ドラマ性）が高いほど、観客のポジティブ感情が持続し、スタジアム周辺のローカルエリアへの回遊距離を伸ばす。

## Methods（方法）
2.1 研究デザイン

本研究は、QGISを用いた空間分析によって、試合の興奮度（独立変数）と試合終了後30分間の移動距離・回遊エリア（従属変数）の関連性を評価する。

### 独立変数（試合のドラマ性）
- 点差
- 逆転回数
- サヨナラの有無
- イニングごとのスコア推移から算出する興奮度指数

### 従属変数
- 試合終了後1時間の平均移動距離
- 滞留地点の属性（飲食店密度、ローカル店かチェーン店か）
- 回遊エリアの広がり（差分ヒートマップによる可視化）

2.2 データ選定と条件統制

信頼性の高い比較を行うため、次のような厳密な条件統制を行う。
- 天候条件の統一：晴天の日に限定
- 曜日条件の一致：平日ナイターに限定
- 終了時刻が近い試合同士を比較
ノイズ除去：スタジアム近隣ユーザーのうち、試合観戦者とみなせるIDに限定し、通勤・住民データを除外
比較対象は 「劇的勝利の日」 vs 「大敗の日」 のペアとして抽出する。

2.3 空間解析プロセス（QGIS）
① 静的ポテンシャルの可視化
QuickOSM によりスタジアム周辺の飲食店（restaurant/pub）を取得
ヒートマップ化し「祝杯の受け皿」となるスポットの空間分布を可視化

② 動的フローのPoC（技術検証）
合成GPSデータを用い、時系列コントローラによる移動再現を実施
動的分析が可能であることを事前に確認

③ 本分析：差分ヒートマップ
「勝ち日の人流ヒートマップ」−「負け日のヒートマップ」＝純増エリア
試合の興奮度によって増えるエリアを特定
